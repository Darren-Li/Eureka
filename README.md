## 📖 Project Overview

- Eureka is an agent-driven analytics platform that enables users to upload data and define analysis goals. It autonomously interprets user intent, decomposes analytical tasks, generates and executes code, and delivers structured, visually rich, and professional reports—automating the entire journey from data to insights.

---

## ✨ Features

- 📂 Feature 1: Data Management
- ⚠️ Feature 2: Configure Your LLMs
- 👥 Feature 3: Auto Data Analysis
- 📈 Feature 4: Return HTML report

---

## ⚙️ Installation and Usage

clone the project
```
git clone https://github.com/Darren-Li/Eureka.git
cd Eureka
```

Requirements
```python
pip install -r requirements.txt
```

Run the Eureka app
```
streamlit run Eureka.py
```
